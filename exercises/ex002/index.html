Paragraphs and breaking space

Ye can write a paragraph in whatever way ye want, as log as it's within the 'p' tags.

If ye need to create a breaking space, then all ye got to do is use the <br> tag. In the following example, ye can see the tag <br> creating a breaking space:

London Brigde is falling down
Falling down, falling down
London Bridge is falling down
My fair lady
