It's imperative to memorise the correct tag and syntax to solve this sort of issue. First of all we got to remember that everything must be wrapped up inside the <picture> tag. Next we create the "default" image(the largest one) by using the <img> tag and we select the image that's going to be used. Then, above this line of code, we use the <source> tag and use the parameter "max-width" and set it with the pixels size of the default image and set the image(the medium one) that's going to be shown instead if the browser cannot display the default one inside the "srcset" parameter. Finally we set the smallest image, and all we do is repeat the previous step: inside the "max-width" parameter we set the pixel size of the medium image and then, inside the "srcset" parameter, we set the smallest image to be displayed if the user's browser cannot show the medim-sized image. We can use as many pictures as necessary to make the page more user-friendly.
