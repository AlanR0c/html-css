Working With Lists

Ordered Lists

The HTML <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical. An ordered list starts with the <ol> tag. Each list item starts with the <li> tag. The list items will be marked with numbers by default.

  1. Wake up
  2. Wash up
  3. Have breakfast
  4. Get ready
  5. Call clients

Ye can also add some parameters to the ordered list. For example, if ye want to have letters instead of numbers or start the list from a specific number or letter.

Ye can use capital or small letters. All ye got to do is choose the letter on "type". Using capital letters:

  1. Wake up
  2. Wash up
  3. Have breakfast
  4. Get ready
  5. Call clients

Using small letters:

  1. Wake up
  2. Wash up
  3. Have breakfast
  4. Get ready
  5. Call clients

Starting the list from a specific number:

  1. Wake up
  2. Wash up
  3. Have breakfast
  4. Get ready
  5. Call clients

Ye can also use Roman characters, either capital or small:

  1. Wake up
  2. Wash up
  3. Have breakfast
  4. Get ready
  5. Call clients
  1. Wake up
  2. Wash up
  3. Have breakfast
  4. Get ready
  5. Call clients
