The <pre> tag defines preformatted text. Text in a <pre> element is displayed in a fixed-width font, and the text preserves both spaces and line breaks.

Abbreviations

The <abbr> tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM". Tip: Use the global title attribute to show the description for the abbreviation/acronym when you mouse over the element. Eg:

HTML5 and CSS3 are widely used front-end languages.

The WHO was founded in 1948.

Overriding Text Direction

BDO stands for Bi-Directional Override. The <bdo> tag is used to override the current text direction. The attribute 'dir'(direction) is required and it specifies the text direction of the text inside the <bdo> element. 'ltr' stands for "left to the right", and 'rtl' stands for "right to the left". Eg:

.nwod gnillaf si egdirB nodnoL

London Bridge is falling down.
